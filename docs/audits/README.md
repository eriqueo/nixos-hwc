# `docs/audits/` — point-in-time audit artifacts

This directory holds **audit deliverables** generated by the nightly-builds
gauntlet and by ad-hoc human investigations. Each subdirectory is a single
audit topic; its contents (scripts, checklists, reports) are reviewable on
their own.

The artifacts here describe **the state of the system at a point in time** and
the **proposed remediation** for what that state revealed. They are not part
of the live nixos configuration — nothing in this tree is imported by any
module — and none of the scripts are executed automatically.

## Law 12 — Structure

| Path | Purpose |
| ---- | ------- |
| `mnt-hot/reconcile.sh` | Dry-run-by-default script that consolidates stray media from `/mnt/hot` orphans into `/mnt/media` and prunes verified-empty orphan dirs. Guarded against the self-derived active-paths set. |
| `mnt-hot/reconcile-checklist.md` | Human run procedure for `mnt-hot/reconcile.sh`. |

## Running an audit script

Every script in this tree honors `DRY_RUN=1` as the default. Always read the
companion checklist (`*-checklist.md`) before flipping `DRY_RUN=0`.

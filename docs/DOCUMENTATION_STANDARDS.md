# Documentation Standards — pointer

old links don't dangle:

- **Per-domain documentation contract**: `CHARTER.md` **Law 12** (README with
  Purpose / Boundaries / Structure / Changelog; updated with every commit that
  touches the domain). Template: `docs/templates/DOMAIN_README_TEMPLATE.md`.
- **Cross-domain docs layout and rules**: `docs/README.md` (single-copy
  archive rule, scripts-are-not-docs, AI-output-is-ephemeral, no category
  forks).

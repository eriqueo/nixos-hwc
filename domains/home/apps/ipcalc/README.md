# ipcalc

## Purpose
Installs the ipcalc IP/subnet calculator CLI as a one-package Home Manager app. Generated by the shared `domains/lib/mkSimpleApp.nix` factory (Law 2: name = folder).

## Boundaries
- ✅ Manages: `hwc.home.apps.ipcalc.enable` → `pkgs.ipcalc` on `home.packages`.
- ❌ Does not manage: any configuration — ipcalc is a stateless CLI.

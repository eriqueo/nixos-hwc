# google-cloud-sdk

## Purpose
Installs the Google Cloud SDK (`gcloud` et al.) as a plain package. One-package module generated by `domains/lib/mkSimpleApp.nix`. Enable via `hwc.home.apps.google-cloud-sdk.enable`.

## Boundaries
- ✅ `pkgs.google-cloud-sdk` in `home.packages` — nothing else.
- ❌ No gcloud auth, projects, components, or config (`~/.config/gcloud` is user-managed).

## Structure
- `index.nix` — three-line `mkSimpleApp` call (name, description, package).

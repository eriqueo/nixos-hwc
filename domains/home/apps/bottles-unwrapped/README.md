# bottles-unwrapped

## Purpose
Installs the Bottles Wine manager (unwrapped variant) as a plain package. One-package module generated by `domains/lib/mkSimpleApp.nix`. Enable via `hwc.home.apps.bottles-unwrapped.enable`.

## Boundaries
- ✅ `pkgs.bottles-unwrapped` in `home.packages` — nothing else.
- ❌ No Wine prefixes, runners, or Bottles configuration; no desktop entry beyond what the package ships.

## Structure
- `index.nix` — three-line `mkSimpleApp` call (name, description, package).

# localsend

## Purpose
Installs LocalSend, the cross-platform LAN file-sharing app, as a one-package Home Manager app. Generated by the shared `domains/lib/mkSimpleApp.nix` factory (Law 2: name = folder).

## Boundaries
- ✅ Manages: `hwc.home.apps.localsend.enable` → `pkgs.localsend` on `home.packages`.
- ❌ Does not manage: firewall openings for LocalSend's ports (system networking domain) or in-app settings.

## Structure
- `index.nix` — mkSimpleApp call: name, description, package selector.

# markitdown

## Purpose
Installs markitdown (`pkgs.python3Packages.markitdown`), a CLI that converts PDF/DOCX/XLSX/images/audio to Markdown. One-package Home Manager app generated by the shared `domains/lib/mkSimpleApp.nix` factory (Law 2: name = folder).

## Boundaries
- ✅ Manages: `hwc.home.apps.markitdown.enable` → the markitdown package on `home.packages`.
- ❌ Does not manage: any configuration or pipelines that consume it — it is a stateless converter CLI.

## Structure
- `index.nix` — mkSimpleApp call: name, description, package selector.

# obsidian

## Purpose
Installs the Obsidian note-taking app as a one-package Home Manager app. Generated by the shared `domains/lib/mkSimpleApp.nix` factory (Law 2: name = folder).

## Boundaries
- ✅ Manages: `hwc.home.apps.obsidian.enable` → `pkgs.obsidian` on `home.packages`.
- ❌ Does not manage: vaults or vault sync (the brain vault at `~/900_vaults/brain` is synced by its own git/Syncthing machinery), plugins, or `.obsidian/` settings.

## Structure
- `index.nix` — mkSimpleApp call: name, description, package selector.

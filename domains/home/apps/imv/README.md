# imv

## Purpose
Installs the imv image viewer as a one-package Home Manager app. Generated by the shared `domains/lib/mkSimpleApp.nix` factory (Law 2: name = folder).

## Boundaries
- ✅ Manages: `hwc.home.apps.imv.enable` → `pkgs.imv` on `home.packages`.
- ❌ Does not manage: imv configuration, keybinds, or default-application/MIME associations.

## Structure
- `index.nix` — mkSimpleApp call: name, description, package selector.

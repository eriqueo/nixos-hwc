## What Is the 3/4/5 Rule for Decks?

The 3/4/5 rule helps ensure stair safety and proper deck layout. For every set of steps, the rise and run must align so that three squared plus four squared equals five squared (3² + 4² = 5²). In practice, this means measuring 3 feet in one direction, 4 feet perpendicular to that, and confirming the diagonal is exactly 5 feet. This creates a perfect right angle.

Local inspectors often flag stair issues first, so proper layout matters. Whether you're working on decks in Bozeman or managing remodeling in Bozeman, MT that involves structural changes, precise measurements prevent costly rework.
